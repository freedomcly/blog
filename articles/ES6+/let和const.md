# let和const

| **项目** | **var** | **let** | **const** |
| :--- | :--- | :--- | :--- |
| 作用域 | 函数作用域 | 块级作用域 | 块级作用域 |
| 变量提升 | 有 | 暂时性死区 | 暂时性死区 |
| 重复声明 | 能 | 不能 | 不能 |
| 重新赋值 | 能 | 能 | 不能 |
| 是否是顶层对象（如window）的属性 | 是 | 不是 | 不是 |
| 执行速度 | 慢 | 比var快65% | 比var快65% |

ES6变量声明的六种方法：

* var
* function
* let
* const
* import
* class
