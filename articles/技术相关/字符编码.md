# 字符编码

## 字符集和字符编码

| **字符集** | **字符编码** | **内容** | **存储** |
| :--- | :--- | :--- | :--- |
| ASCII | ASCII | 英语和西欧常用字符，共128个字符 | 1个字节来表示绰绰有余 |
| EASCII |  | 扩展到欧洲各国字符 | 1个字节（256个字符） |
| GBK |  | 汉字 | 每个字符可以由1、2、4个字节组成 |
| 重新赋值 | 能 | 能 | 不能 |
| 是否是顶层对象（如window）的属性 | 是 | 不是 | 不是 |
| 执行速度 | 慢 | 比var快65% | 比var快65% |
| Unicode | utf-8<br/>utf-16<br/>utf-32 | 块级作用域 | 块级作用域 |
