# 字符编码

## 字符集和字符编码

| **字符集** | **字符编码** | **内容** | **存储** |
| :--- | :--- | :--- | :--- |
| ASCII | ASCII | 英语和西欧常用字符，共128个字符 | 1个字节来表示绰绰有余 |
| EASCII | EASCII | 扩展到欧洲各国字符 | 1个字节（256个字符） |
| GBK（GB2312、GB18030等） | GBK | 汉字 | 每个字符可以由1、2、4个字节组成 |
| Unicode | UTF-8<br/>UTF-16<br/>UTF-32 | 所有语言 | 1、2、4个字节<br/>2、4个字节<br/>4个字节 |

## Unicode

Unicode是一个包含所有语言的字符集标准，规定了每个字符对应的code point（如\u00e1）。Unicode的范围是ux0000 ~ ux10FFFF，分成17个平面。其中第一个平面，也就是范围为u+0000 ~ u+FFFF的65536（16<sub>4</sub>）个code point包含大部分常用字符，即Basic Multilingual Plane——BMP。
