# 字符编码

## 字符集和字符编码

| **字符集** | **字符编码** | **内容** | **存储** |
| :--- | :--- | :--- | :--- |
| ASCII | ASCII | 英语和西欧常用字符，共128个字符 | 1个字节来表示绰绰有余 |
| EASCII | EASCII | 扩展到欧洲各国字符 | 1个字节（256个字符） |
| GBK（GB2312、GB18030等） | GBK | 汉字 | 每个字符可以由1、2、4个字节组成 |
| Unicode | UTF-8<br/>UTF-16<br/>UTF-32 | 所有语言 | 1、2、4个字节<br/>2、4个字节<br/>4个字节 |
