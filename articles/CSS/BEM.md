# BEM

    .block__element--modifier
    
BEM是源自俄罗斯团队Yandex的前端方法论。

* B--block
* E--element
* M--modifier

## 规则

* block可以嵌套
* 页面中block名称是唯一的
* 每个block范围内element名称是唯一的
* block是独立的，不应与包含块耦合，可以移动到任意其他位置
