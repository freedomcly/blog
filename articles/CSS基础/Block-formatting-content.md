# Block-formatting-context

# 什么是BFC？

Block formatting context即块级格式化上下文。在CSS 2.1中，定位模式有三种：
* normal flow
* floats
* absolute positioning

BFC属于normal flow（正常流），一个普通的html即是一个BFC。
