# 变量提升和函数声明提升

## 变量提升

    // Uncaught ReferenceError: a is not defined
    console.log(a)

    // undefined
    console.log(b)
    var b

    // undefined
    console.log(c)
    var c = 1
    
## 函数声明提升

    // fool
    foo()

    function foo () {
      console.log('fool')
    }

变量提升和函数声明提升的区别：函数声明提升可以提前获得函数体，变量提升只能获取变量在之后会被声明这个信息，无法提前获取真实值，在赋值前一直是undefined。

**同名时函数优先**。
